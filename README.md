"Fix Karachi" – A Civic Issue Reporting Mobile App using React Native & TypeScript.
The idea was to create a simple platform where citizens can report real civic issues like broken roads, water leakage, and street problems — and track their resolution in real time.
💡 What we built:
A mobile app focused on improving civic awareness and community reporting through technology.
⚙️ Tech Stack:
React Native • TypeScript • Context API • React Navigation
🔥 Key Features:
Report civic issues with images
Community voting system
Issue status tracking (Pending / In Progress / Resolved)
Clean and responsive UI
User authentication system
